# Example Work Breakdown Structure Data

Example Work Breakdown Structure Data

## Usage

``` r
wbs_table
```

## Format

A data frame with columns:

- id:

  unique key for each row

- pid:

  parent key for each row

- name:

  character name of the element

- work:

  percent of total work for this element

- budget:

  budget for this element
